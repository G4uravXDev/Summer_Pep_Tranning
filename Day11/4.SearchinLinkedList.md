```python
# Problem Name : Search In LinkedList
# Link : http://geeksforgeeks.org/problems/search-in-linked-list-1664434326/1
# Approach : Linked List

def searchInLinkedList(head, x):
    current = head

    while current is not None:
        if current.data == x:
            return 1
        current = current.next

    return 0
```
